Clone the repository:-
->git clone https://github.com/yash2354-ch/Food-order.git
-> cd Food-order

Install dependencies for server:-
-> cd server
-> npm install

Install dependencies for client:-
-> cd ../client
->npm install

Run the application
Open two terminals:
**backend:**
-> cd server
-> npm start
**Frontend**
->cd client
->npm start

Visit the website
-> http://localhost:3000
